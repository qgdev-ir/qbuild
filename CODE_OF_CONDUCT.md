# QBuild Code Of Conduct
These are rules set to improve communication between contributors of the project \
if you saw violation of rules mention us there

## Rules
### 1. Be polite and kind
To make a happy and friendly community
### 2. Don't publish unrelated to project (advertisement, etc.)
This community is only for developing and maintaining the project so any unrelated content is forbidden
### 3. Don't speak in any language other than English
We assume everyone here know english and speaking in other languages makes confusion
